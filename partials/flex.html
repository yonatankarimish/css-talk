Flex Display

Property Meaning
flex-direction Specifies the direction in which flex elements are laid out.
flex-wrap Defines the way flex items behave when wrapping is needed.
justify-content Handles the alignment of flex items along the main axis.
align-items Handles the alignment of flex items along the cross-axis
Property Meaning
order A number which states the order in which the items are laid out.
flex-grow A proportion by which to modify the item's size.
align-self Custom alignment for a flex item, to override align-items of the flex-container.
