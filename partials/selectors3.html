Selectors

Selector Meaning
element[attribute] Applied to elements which have the specified attribute.
element[attribute='value'] Applied to elements when the specified attribute matches the provided value.
:first-child Applied to elements which are the first child of their parent.
:not(selector) Negates the expression inside the brackets.
:nth-child(keyword) Can be used with "odd", "even" or a number.
input:valid Selects inputs which are valid.
:checked Selects checkboxes which are checked.
